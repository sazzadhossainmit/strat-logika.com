## Project Overview

Strat Logika is a responsive corporate website built using **HTML, CSS, and JavaScript**.  
It presents industrial services across renewable energy, waste management, drilling operations, and vessel supplies.

The website is fully functional, responsive, and production-ready.

---

## Core Features

- Responsive Desktop & Mobile Navigation  
- Mega Menu (Services Section)  
- Fullscreen Hero Image Slider  
- Tab-Based Service Layout  
- Filterable Project Portfolio  
- Contact Modal with Formspree Integration  
- Clean UI with Inter Font & Lucide Icons  

---

## Technology Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- Google Fonts (Inter)  
- Lucide Icons  
- Formspree (Contact Form Handling)
